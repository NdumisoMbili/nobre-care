# Nobre Care | The Future of Emergency Management

A premium, high-fidelity emergency response platform designed for the complex needs of modern urban safety. Nobre Care unifies ambulances, security, fire rescue, and aerial drone response into a single, high-density interface.

## Features

- **Unified Network**: Instant connection to multiple emergency branches (E-Hailing Ambulances, Security, Fire Rescue).
- **Aerial Response**: Cutting-edge drone integration for rapid situational assessment and medical delivery.
- **App Ecosystem**: A side-by-side high-performance mobile layout for iOS and Android.
- **Live Tracking**: Real-time tracking of responding units with glassmorphic UI elements.
- **Premium Identity**: Sophisticated technical design with cinematic "blood red" and navy aesthetics.

## Tech Stack

- **Structure**: Semantic HTML5
- **Styling**: Vanilla CSS3 (Custom Grid/Flexbox architectures)
- **Logic**: Vanilla JavaScript (Intersection Observer, magnetic buttons, staggered animations)
- **Icons**: FontAwesome 6.4

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/NdumisoMbili/nobre-care.git
   ```
2. Open `nobre-care.html` in any modern web browser.
